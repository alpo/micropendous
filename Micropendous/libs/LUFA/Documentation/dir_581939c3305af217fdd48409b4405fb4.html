Board specific LED driver header for the BLACKCAT USB JTAG.


Detailed Description

This folder contains drivers for hardware on the TCNISO Blackcat USB JTAG (http://www.embeddedcomputers.net/products/BlackcatUSB/. The header files in this folder should not be included directly in user applications; the similarly named dispatch header files located in the parent Board directory should be included instead.
